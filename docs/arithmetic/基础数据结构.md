## 一、Trie

Trie 树，又叫字典树、前缀树（Prifix Tree）、单词查找树 或 键树，是一种多叉树结构。

字典树的性质

1. 根结点（Root）不包含字符，除根节点外的每一个节点都仅包含一个字符
2. 从根节点到某一节点路径上所经过的字符连接起来，即为该节点对应的字符串；
3. 任意节点的所有字节点所包含的字符串都不相同

Use cases : 自动补全、拼写检查、IP 路径（最长前缀匹配）、T9（九宫格）打字预测、词频统计（节省内存）
